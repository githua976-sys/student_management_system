# Student Management System

## Project Overview

The Student Management System is a Python-based application designed to manage student records and course enrollment information. The system allows administrators to add, view, search, update, and delete student records while storing data securely in an SQLite database.

This project was developed as an End of Module Python Project to demonstrate the use of Python programming, Object-Oriented Programming (OOP), database management, and CRUD operations.

---

## Problem Statement

Managing student records manually can be time-consuming and prone to errors. Educational institutions require a reliable system for storing and managing student information efficiently.

The Student Management System provides a digital solution for managing student records and course enrollment.

---

## Objectives

- Add new student records
- View all student records
- Search for a student
- Update student information
- Delete student records
- Store student data permanently using SQLite
- Manage student course enrollment
