#  Student Management System

##  Project Overview

The **Student Management System** is a desktop application developed using **Python**, **Tkinter**, and **SQLite**. It provides an easy way for educational institutions to manage student records, courses, enrollments, attendance, and reports through a graphical user interface (GUI).

The project demonstrates the practical application of Python programming, database management, CRUD operations, and GUI development.

---

#  Objectives

The project aims to:

* Manage student records efficiently.
* Manage course information.
* Enroll students into different courses.
* Record and monitor student attendance.
* Generate summary reports.
* Store information securely using SQLite.
* Provide a simple and user-friendly graphical interface.

#  Installation

1. Clone the repository

```
git clone https://github.com/yourusername/student_management_system.git
```

2. Open the project folder

```
cd student_management_system
```

3. Install dependencies

```
pip install -r requirements.txt
```

4. Run the application

```
python gui.py
```
